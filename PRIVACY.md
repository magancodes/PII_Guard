# Privacy Policy for PII Guard

## 1. Information Collection
**PII Guard** does not collect, store, or transmit any standard personally identifiable information to external servers by default. All regular expression-based scanning (for Emails, Phone Numbers, and UPI IDs) happens entirely locally within your browser.

## 2. Gemini API Usage (Optional)
If you choose to enable the **Advanced Name Detection** feature, the extension will send text snippets from the webpage to the Google Gemini API using the API Key you provide. 
- The data is sent directly from your browser to Google's API servers.
- The extension developer does not have access to this data or your API key.
- Please refer to Google's API Privacy Policy for details on how they handle data processed by their APIs.

## 3. Data Storage
Local settings, such as your whitelist, selected PII types to hide, and your optional Gemini API key, are stored locally using Chrome's `chrome.storage.sync` or `chrome.storage.local` API. This synced data is protected by your Google Account security if you have Chrome Sync enabled.

## 4. Third-Party Services
We do not sell, trade, or otherwise transfer your information to outside parties.

## 5. Contact
For any questions regarding this privacy policy, please contact the developer via the official web store listing.
